# Linux DevOps Project

## Project Overview

This project demonstrates a complete DevOps workflow using AWS and Linux.

## Technologies Used

- AWS EC2
- AWS RDS MySQL
- Apache HTTP Server
- PHP
- Linux Shell Scripting
- Cron Jobs
- Git
- GitHub

## Features

- Hosted a website on EC2
- Connected PHP application to RDS MySQL
- Automated server health checks using shell scripts
- Scheduled jobs using Cron
- Version control using Git and GitHub
- Automated deployment using deploy.sh

## Project Workflow

Developer
↓
GitHub Repository
↓
EC2 Server
↓
Apache Web Server
↓
Public Website
